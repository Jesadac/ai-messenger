# AI Messenger User Guide

## Getting started

1. Start Ollama, then open Tools → Setup local model. Use automatic detection or enter the local server address and choose Save & Connect.
2. Open Help → Installed Model Guide to see each installed model's size and capabilities.
3. Click a contact to open a chat. In a one-to-one chat, use the model picker to choose the local model.
4. Use Tools → About Me to set your name, picture, background, goals, work style, and guidance preferences.
5. Use the AI profile menu or right-click a contact to set its name, picture, tone, gender, Soul, personality, and skills.
6. In the main contact list, figurine markers identify capabilities: glasses for OCR or vision, a paintbrush for image generation, and a lightbulb for reasoning models. The timestamp shows the most recent message; after a day passes it includes the date and time.
7. Type a message and press Send or Enter. Your message appears immediately; Stop ends active generations. Thinking controls Fast or Deep response effort where supported.
8. Use Send Files for images or text files, and Share view for a user-selected desktop or window snapshot. Only explicitly attached content is sent to the local model.
9. Use Invite to create a conversation with multiple assistants. In a one-to-one chat, Invite creates a new group. In an existing group, Invite adds or removes assistants from that same group and retains its name and saved history; messages from removed assistants remain visible with gray names. In a team that includes Chief of Staff, address the Chief with the desired outcome, such as “interpret this poem,” and attach the source. Chief of Staff identifies the right specialists, delegates the work in the shared thread, and synthesizes their real replies. You do not need to micromanage each assistant.
10. Use the chevron to attach the chat to the contact list. Use Detach to give it its own window; other model tasks continue in the background.
11. File → New Chat starts a fresh visible session while saved history remains available through Chat History. Loading old history can add outdated context.
12. Unused assistants suspend after five minutes and show a clock icon; after ten minutes they move Offline. Click the assistant to wake it. Right-click an assistant to suspend or wake it manually.
13. Use Actions for light/dark mode and Tools for font size, setup, and assistant controls. Help contains the model guide, features, setup guide, license, terms, and user-data information.
14. To permanently remove AI Messenger data on Windows, quit the app, open `%APPDATA%\Retro Messenger Assistant` in File Explorer, and delete that folder. This removes the encrypted vault, chats, profiles, and settings.
