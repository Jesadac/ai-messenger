# AI Messenger Quick Start

## Before you begin

1. Install Ollama and download at least one local model.
2. Start Ollama.
3. Open AI Messenger.

## Connect your local model

1. Open **Tools → Setup local model**.
2. Choose **Detect Ollama automatically**, or enter the local server address.
3. Select **Test Connection**, then **Save & Connect**.
4. Open **Help → Installed Model Guide** to review the available models and their capabilities.

## Create your first assistant

1. Select **+** in the contact list.
2. Choose an installed model.
3. Enter a display name and optionally choose an image.
4. Choose a personality or role preset, or add your own Soul, personality, and skills instructions.
5. Save the profile, then click it to open a chat.

## Use a chat

- Type a message and select **Send**. Your message appears immediately.
- Use **Send Files** to attach an image or text file.
- Use the snapshot control to attach a selected screen or application window.
- Use **Stop** or **Stop All** to end active responses.
- Use **File → New Chat** for a fresh visible conversation. Earlier threads remain available from **Tools → This Chat’s History**.

## Work with a group

1. Create a group from the contact list or invite another assistant from a one-to-one chat.
2. Use `@Name` in the message field to direct a request to one participant while keeping the discussion in the group.
3. Select **Invite** in an existing group to add or remove participants. Existing messages are retained; former participants are shown with gray names in the transcript.

## Personalize the app

- **Tools → About Me** sets shared information used for personalization across chats.
- Use an assistant’s profile controls to change its image, profile settings, and model.
- **Actions** provides light and dark themes. **Tools** provides text-size settings.
