## What the application is

AI Messenger is a local desktop interface layer for Ollama or another compatible local inference server. It helps a user interact with models installed on that user's device. It is not Ollama, is not a hosted AI service, and does not grant rights to any model or third-party software.

## Local processing and privacy

The application is designed not to send data or usage reports to the developer. It contains no developer telemetry, analytics, advertising, or remote account service. Chat messages, attachments, profiles, settings, and history are stored locally on the user's device. Requests are sent to the local inference endpoint selected by the user. A screen capture or file is sent to that local endpoint only when the user explicitly attaches it.
